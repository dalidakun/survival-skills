# 如何在 GitHub 查看文件内容

## 步骤 1：打开 GitHub 仓库

1. **打开浏览器**（Chrome、Edge 等）
2. **访问这个地址**：
   ```
   https://github.com/dalidakun/survival-skills
   ```
3. 如果提示登录，使用您的 GitHub 账号登录

## 步骤 2：进入 content 文件夹

1. 在仓库页面，您会看到文件和文件夹列表
2. **找到并点击 `content` 文件夹**（应该能看到这个文件夹）
3. 点击后进入 `content` 文件夹

## 步骤 3：查看最新文件

1. 在 `content` 文件夹中，您会看到很多 `.md` 文件
2. **找到最新的文件**（通常在最上面，或者按时间排序）
3. **点击文件名**，打开文件内容

## 步骤 4：检查 updatedAt 字段

打开文件后，您会看到文件内容，开头应该是这样的：

```yaml
---
slug: xxx
title: xxx
subtitle:
image:
link: https://...
updatedAt: "2024-12-19T10:30:00.000Z"
intro: ""
---
```

**检查重点**：
- 找到 `updatedAt:` 这一行
- 看后面是否有**引号**（双引号 `"`）
- **有引号**：`updatedAt: "2024-12-19T10:30:00.000Z"` ✅ 正确
- **没有引号**：`updatedAt: 2024-12-19T10:30:00.000Z` ❌ 可能有问题

## 步骤 5：如果看到没有引号的文件

如果看到旧文件没有引号，可以：

### 方法 1：重新提交文章（推荐）
1. 在网站管理后台
2. 找到这篇文章
3. 重新提交一次（会更新文件并加引号）

### 方法 2：手动编辑（如果熟悉 GitHub）
1. 在文件页面点击右上角的**铅笔图标**（Edit）
2. 找到 `updatedAt:` 这一行
3. 在时间值前后加上引号：`updatedAt: "2024-12-19T10:30:00.000Z"`
4. 滚动到页面底部
5. 点击 "Commit changes"
6. 填写提交信息，点击 "Commit changes"

## 快速检查方法

**最简单的方法**：
1. 访问：https://github.com/dalidakun/survival-skills/tree/main/content
2. 点击最新的 `.md` 文件
3. 查看文件开头，找到 `updatedAt:` 这一行
4. 看是否有引号
