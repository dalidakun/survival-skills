## 详细步骤：七牛云 + 免费域名

### 步骤 1：获取免费域名

**选项 A：使用 Freenom（完全免费）**

1. 访问：https://www.freenom.com/
2. 注册账户
3. 搜索并注册免费域名（如：`yourname.tk`）
4. 免费使用 1 年，可续期

**选项 B：注册便宜域名**

1. 访问域名注册商（如：Namecheap、GoDaddy）
2. 注册 .xyz 域名（约 $1/年）
3. 非常便宜，几乎等于免费

### 步骤 2：在七牛云绑定自定义域名

1. 登录七牛云控制台
2. 进入存储空间 → "域名管理"
3. 点击"绑定域名"
4. 输入子域名（如：`pdfs.yourname.tk`）
5. 获取 CNAME 记录值

### 步骤 3：配置 DNS

1. 在域名管理面板（Freenom 或域名注册商）
2. 添加 CNAME 记录：
   - 主机记录：`pdfs`
   - 记录类型：`CNAME`
   - 记录值：七牛云提供的 CNAME 地址
3. 等待 DNS 生效（通常几分钟到几小时）

### 步骤 4：配置环境变量

```
QINIU_DOMAIN=https://pdfs.yourname.tk
```

## 详细步骤：Cloudflare R2 + 自定义域名

### 步骤 1：获取域名

（同上，使用 Freenom 或注册便宜域名）

### 步骤 2：在 Cloudflare 添加域名

1. 登录 Cloudflare Dashboard
2. 点击 "Add a Site"
3. 输入您的域名
4. 选择免费计划
5. 按照提示配置 DNS

### 步骤 3：在 R2 绑定自定义域名

1. 进入 R2 存储桶设置
2. 找到 "Custom Domains"
3. 添加自定义域名（如：`pdfs.yourname.tk`）
4. Cloudflare 会自动配置 SSL 和 DNS

### 步骤 4：配置环境变量

```
CLOUDFLARE_R2_PUBLIC_URL=https://pdfs.yourname.tk
```
