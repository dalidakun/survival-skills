# 调试更新标签问题

## 问题
1. 新文章没有显示"更新"标签
2. 新文章没有排在最前面

## 可能的原因

### 1. 时区问题
- `updatedAt` 是 UTC 时间（ISO 格式）
- 判断"今天"使用的是服务器本地时间
- 如果时区不同，可能导致判断错误

### 2. 数据未正确解析
- `updatedAt` 字段可能没有正确读取
- 或者格式不正确

## 调试方法

### 检查文件内容
查看 GitHub 仓库中新创建的文件，确认 `updatedAt` 字段是否存在：
- 访问：https://github.com/dalidakun/survival-skills
- 查看 `content/` 文件夹中的最新文件
- 检查 frontmatter 中是否有 `updatedAt: 2024-XX-XX...`

### 检查时区
- Vercel 服务器可能使用 UTC 时区
- 您的本地时间可能是其他时区
- 需要确保时区一致

## 临时解决方案

如果问题持续，可以：
1. 检查新创建的文件是否有 `updatedAt` 字段
2. 手动编辑文件，添加正确的 `updatedAt` 时间
3. 或者等待代码更新后重新创建文章
