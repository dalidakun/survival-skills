# 七牛云 CDN 域名配置填写说明

## 页面说明

您当前在 **CDN 域名管理** 页面，这是正确的配置方式。七牛云可以通过 CDN 来加速访问存储空间的文件。

## 填写说明

### 1. 基本信息 (Basic Information)

#### 加速域名 (Accelerated Domain Name)
**填写**：`pdfs.okapdf007.asia`

**注意**：
- 如果域名已备案，直接填写
- 如果域名未备案，可能需要先备案（.asia 域名通常需要备案）

#### 通信协议 (Communication Protocol)
**选择**：`HTTPS`（推荐）或 `HTTP`

**建议**：
- 优先选择 **HTTPS**（更安全，现代浏览器推荐）
- 如果选择 HTTPS，七牛云会自动申请免费 SSL 证书

#### 覆盖范围 (Coverage Area)
**选择**：`中国大陆`

**说明**：
- 如果用户主要在中国，选择"中国大陆"
- 如果用户遍布全球，选择"全球"

#### 使用场景 (Usage Scenario)
**选择**：`下载分发`

**说明**：
- PDF 文件属于下载分发类型
- 这个选项适合文件下载场景

#### IP 协议 (IP Protocol)
**选择**：`IPv4 / IPv6` 或 `IPv4`

**建议**：
- 选择 `IPv4 / IPv6`（支持更多访问方式）
- 如果只支持 IPv4，选择 `IPv4`

### 2. 源站配置 (Origin Server Configuration)

#### 源站配置 (Origin Server Configuration)
**选择**：`七牛云存储`

**说明**：
- 选择"七牛云存储"表示从您的存储空间获取文件
- 这是正确的选择

#### 存储空间选择
**选择**：`survival-skills-pdfs`（您的存储空间名称）

**说明**：
- 从下拉菜单中选择您创建的存储空间
- 如果看不到，确认存储空间已创建


## 重要提示

### 1. 域名备案

**重要**：`.asia` 域名在中国使用可能需要备案。

**如果提示需要备案**：
1. 需要先完成域名备案
2. 备案通常需要 7-20 个工作日
3. 备案期间可以先使用 HTTP 协议测试

**如果不想备案**：
- 可以考虑使用已备案的域名
- 或者先使用七牛云的测试域名（30天有效期）

### 2. HTTPS 证书

如果选择 HTTPS：
- 七牛云会自动申请免费 SSL 证书
- 证书申请通常需要几分钟到几小时
- 申请完成后，域名就可以使用 HTTPS 访问

### 3. CDN 回源流量费

**注意**：页面提示"当源站选择存储时,若发生回源请求将会由存储产生 CDN 回源流量费"

**说明**：
- 文件首次访问时会从存储空间回源
- 之后会缓存在 CDN 节点
- 回源流量会产生少量费用（通常很小）
- 免费额度内通常不会产生费用

## 填写完成后

1. **点击"确定"或"提交"**
2. **等待配置生效**（通常几分钟）
3. **配置 DNS 记录**（如果需要）
4. **在 Vercel 配置环境变量**

## 配置 DNS（如果需要）

如果七牛云提示需要配置 DNS：
1. 在您的域名管理面板添加 CNAME 记录
2. 主机记录：`pdfs`
3. 记录值：七牛云提供的 CNAME 值
4. 等待 DNS 生效

## Vercel 环境变量配置

配置完成后，在 Vercel 添加环境变量：

```
QINIU_DOMAIN=https://pdfs.okapdf007.asia
```

（如果选择 HTTP，使用 `http://pdfs.okapdf007.asia`）

## 验证配置

配置完成后：
1. 等待几分钟让配置生效
2. 访问：`https://pdfs.okapdf007.asia`（应该能看到七牛云页面或错误页面，说明域名已绑定）
3. 在管理后台上传测试 PDF
4. 检查返回的链接是否为：`https://pdfs.okapdf007.asia/文件名.pdf`

## 常见问题

### Q1: 提示域名需要备案怎么办？

**A**: 
- 如果必须备案，需要先完成备案流程
- 或者先使用 HTTP 协议（不需要备案）
- 或者使用七牛云的测试域名

### Q2: HTTPS 证书申请失败？

**A**:
- 检查域名是否正确
- 检查 DNS 是否已生效
- 等待更长时间（最长可能需要几小时）
- 可以先使用 HTTP 测试

### Q3: 配置后无法访问？

**A**:
- 检查 DNS 记录是否正确
- 等待配置生效（可能需要几分钟到几小时）
- 检查域名是否已备案（如果需要）
